# 📊 Sales Intelligence Dashboard

This is a data analytics dashboard built using Streamlit.

## 🚀 Features
- KPI Metrics (Sales, Profit, Margin)
- Monthly Sales Trend
- Region & Category Filters
- Date Range Filtering

## 🛠 Tech Stack
- Python
- Pandas
- SQL (SQLite)
- Streamlit
- Matplotlib

## 📂 Dataset
Superstore dataset (2014–2017)

## ▶️ Run the project
python create_database.py
streamlit run app.py
